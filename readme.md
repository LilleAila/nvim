# Neovim Config

> [!WARNING]
> I have switched to nixvim. My new config is [here](https://github.com/LilleAila/nvim-nix)

## Linux and MacOS:
#### Backup old config
```bash
mv ~/.config/nvim ~/.config/nvim-old
```
#### Clone this repo
```bash
git clone https://github.com/LilleAila/nvim2 ~/.config/nvim
nvim
```
#### Update config
```bash
cd ~/.config/nvim && git pull
```
